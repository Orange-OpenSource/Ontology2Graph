# Ontologies Folder

This folder is intended to store ontology files used in the project. Ontologies define the structure, relationships, and semantics of the data used by the system.

## Usage
- Place all ontology files relevant to your project in this directory.
- The provided `Noria.ttl` file serves as an example of an ontology. You can use it as a reference.

## Example
- `Noria.ttl`: Example ontology file for the Norai project.

Feel free to add, modify, or remove ontology files as needed for your specific use case.
