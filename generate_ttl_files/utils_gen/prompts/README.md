# Prompts Folder

This folder contain a json file that references several prompts (User prompts and System prompts). Prompts are typically used to guide LLMs in generating specific outputs or behaviors. The file can be used as is or modified accordingly to your own need.

You will find also a prompt to optimize user prompts, that as been used to generate prompts P3 and P3\_1 and a prompt to create user prompt, that as been used to create prompt P4\_1. 

## Usage
- If you use and API, set your LLM API key in an environment variable called `LLM_PROXY_KEY`.
- Place all prompt relevant to your project in this directory.

## Example
- Feel free to add, modify, or remove prompt files as needed for your specific use case.
