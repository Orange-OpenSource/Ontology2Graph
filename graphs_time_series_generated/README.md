# Graph generated at regular interval

In this folder are stored graph generated at regular intervals.  

In order to produce line charts based on kpi produced by robot (http://robot.obolibrary.org/) you have to do the following step

## First : Produce robots results

Robot meaurements are produce thanks to robots_cmd.py. You just have to pass one argument to the python script corresponding to the location folder where are stored the Knoweledge graphs in turtle format generated by graphllm.py. Results are stored accordingly to the {PATH_ROBOT} constant in csv format.

## Second : Generate Time Series 

Time series are generated thanks to build_time_series.py. You just have to pass one argument to the python script corresponding to the location where robot measurement created by robots_cmd.py are stored. Results are stored in a single one filenames accordingly to the {CONCATENATED_FILE_NAME} constant

## Third : Dysplay the line charts

plot_line_charts.py will plot the line chart, the name of the KPIs must be use as a first argument then the location of the result file generated by build_time_series.py as a second argument. Plots are also stored in html format in a folder named "Charts"
